Présentation

Avaya Rooms Systems est une application de visioconférence qui vous permet de rencontrer d'autres personnes en ligne. Pour toucher un public plus large, vous pouvez utiliser Brightcove Live pour diffuser votre réunion en streaming afin que les autres puissent la voir.

Étapes

Suivez ces étapes pour organiser une réunion sur Brightcove Live:

    {% if site.product_short == Video Cloud-fr/ ou site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %}
  1. Dans Studio, créez un événement en direct à l'aide du module Live. {% endif %} {% if site.product_short == Live-fr/ %}
  2. Dans Studio, créez un événement en direct à l'aide du module Live. {% endif %}
    Remarque: Si vous souhaitez enregistrer la diffusion une fois celle-ci terminée, assurez-vous de cocher la case Convertir l'événement en ressource vidéo une fois terminé.™
  3. Une fois que vous avez créé l'événement en direct, copiez l'URL RTMP Streaming et la clé de diffusion depuis le panneau Control Room.
    • URL RTMP de streaming - URL du point de terminaison de streaming (Vous pouvez trouver l'URL du point de terminaison dans la salle de contrôle Brightcove Live
    • Clé de diffusion en continu: nom du flux (dans Brightcove, le nom du flux est toujours actif-fr/)
  4. Publiez et intégrez le lecteur de cet événement en direct sur votre site web/page intranet, et notez l'URL de la page.

Configuration de la session de streaming

Avec l'URL RTMP Streaming et la clé de diffusion en continu, vous devez les configurer sur l'Avaya Codec Unit.

  1. Dans l'interface graphique Avaya, accédez à CONFIGURER > STREAMING.

    (Facultatif) Sur le site Web d'Avaya, accédez à PARAMÈTRES DE L'ADMINISTRATEUR > UTILITAIRES > STREAMING.

  2. Insérez votre URL RTMP Streaming.
  3. Insérez votre clé Streaming.

Démarrer/arrêter la session de streaming

Pour démarrer ou arrêter la session de streaming, procédez comme suit:

  1. Accédez à PAGE D'ACCUEIL > PLUS > STREAMING.
  2. Cliquez sur le bouton Streaming. Cela lancera la diffusion en direct.
  3. Le streaming RTMP est activé par défaut. Pour le désactiver, sur la plate-forme Avaya, allez dans CONFIGURATION > AVANCÉ > UTILITAIRES > STREAMING.
