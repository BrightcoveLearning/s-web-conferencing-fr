Présentation

INXPO est une plateforme de streaming vidéo permettant de créer des expériences événementielles de style télévisuel. Pour toucher un public plus large, vous pouvez utiliser Brightcove Live pour diffuser votre événement en continu.

Étapes

Suivez ces étapes pour organiser une réunion sur Brightcove Live:

  1. Dans Studio, créez un événement en direct {% if site.product_short == Video Cloud-fr/ ou site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %} en utilisant le module Live. {% endif %} {% if site.product_short == Live-fr/ %} événement en direct à l'aide du module Live. {% endif %}
    Si vous souhaitez enregistrer la diffusion une fois qu'elle est terminée, assurez-vous de cocher la case Convertir l'événement en vidéo une fois terminée™ case à cocher !
  2. Une fois que vous avez créé l'événement en direct, copiez l'URL RTMP Streaming et la clé de diffusion depuis le panneau Control Room.
    • URL RTMP de streaming - URL du point de terminaison de streaming (Vous pouvez trouver l'URL du point de terminaison dans la salle de contrôle Brightcove Live
    • Clé de diffusion en continu: nom du flux (dans Brightcove, le nom du flux est toujours actif-fr/)
  3. Publiez et intégrez le lecteur de cet événement en direct sur votre site web/page intranet, et notez l'URL de la page.

Configuration de la session de streaming

Avec l'URL RTMP Streaming et la clé de diffusion en continu, vous devez les configurer sur l'outil de production INXPO.

  1. Accédez à l'outil de production INXPO et cliquez sur l'icône à 3 points pour accéder à la section Détails de la sortie.
  2. Cliquez sur l'icône représentant un crayon.
  3. Remplissez les informations suivantes avec vos informations d'identification pour l'événement Brightcove Live.
  4. Dans le champ Titre, ajoutez le nom du flux en direct de Brightcove.
  5. Dans le champ Profil d'encodage, sélectionnez le profil d'encodage par défaut dont les paramètres définissent la façon dont l'encodeur compresse et diffuse la webdiffusion aux participants.
  6. Dans le champ URL RTMP, insérez l'URL du point de terminaison de streaming Brightcove.
  7. Cliquez sur le bouton OK.
